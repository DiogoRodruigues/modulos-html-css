alguns que tem:

        
            

                Primeiro <h1>
                Segundo <h2>
                terceiro <h3>
                Quarto <h4>

                sexto <h6>

pra onde se aplica cada uma:

<h1> usada em:

É o título principal da página, que deve ser claro, direto e conciso

<h2> usado em:

É usada como suporte, Ajuda a organizar as informações.

<h3> usado em:

É usada para subtítulos de nível inferior, geralmente dentro das seções criadas pelos H2s e ajuda a organizar o conteúdo de forma mais detalhada

<h4>, <h5> e<h6> usaod em:

São usadas para notas, listas, percepções, informações de contato e direitos autorais adicionais
