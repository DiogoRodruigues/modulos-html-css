4 - mark, del e ins

Tag mark:

a tag "Mark" é pra deixar uma cor em especifico em uma palavra como a gente usa com o maca texto de cores diferentes

        
Exemplo:

        
Amigos são como estrelas: você nem sempre vê, mas sabe que estão lá.

        <mark>Amigos são como estrelas: você nem sempre vê, mas sabe que estão lá.</mark>
        
se quiser pode colocar diferentes tipos de cor, basta usar o CSS inline que é ir dentro da tag Mark e colocar "style="background-color: Blue;" e assim muda a cor do mark


        <mark style="background-color: Blue;>Amigos são como estrelas: você nem sempre vê, mas sabe que estão lá.</mark>

Tag del:

a tag del é pra dizer que aquela palavra ta deletada ou errada

        
Exemplo:

        
Lorem ipsum dolor sit amet consectetur adipisicing elit. Odio quibusdam aspernatur maiores ad tempore molestias ab aperiam, optio eligendi modi veniam dolorem vero nemo quisquam, nostrum iste ut incidunt quia?


        
<del>Texto-Usado-De-Exemplo-Para-Resposta-Errada</del>

Tag ins:

a tag ins é para deixar a palavra sublinhada

        
Exemplo:

        
Lorem ipsum dolor sit amet, consectetur adipisicing elit. Quos accusamus, natus iste ducimus facilis modi ullam veritatis laudantium eum fugit error fuga aut illum autem vel. Molestias numquam ipsum sunt.


        
<ins>Palavra-Usada-De-Exemplo-Para-Sublinhar</ins>
